Code for support tags for earlier version of IE

The many latest HTML5 tags are supported in IE9 and next version of IE but older versions of IE not support many HTML5 and CSS3 tags.To overcome this problem use the code retrn below in your programme.

Syntax of conditional statement

[if lt IE 9]
-----------------
-----------------
/* type below code in script tag */
<script type=text/javascript src ="http//html5shiv.googlecode.com/svn/trunk/html5.js">

![endif]

Type above code in comment tags in head sectiion of your markup. In above code 'lt' keyword means less than.And script is used for support html5 tags using googlecode
