Factory function

In javascript there are two type of functions are used for create objects

  • 1.factory function -which creats the objects then return it.
  • 2.constructor -which creats the objects then return it.

Factory function:.

Using factoy method we can create number of same type of objects.Same type of means the objects having same properties.

Syntax:

var name_of_factory_method = function(variables,..,...)
  return{
         .....................;
         };
};


Ex. Create the person factory function which has two arguments and it returns the firstname ,lastname and 'sayhi' method.

var createPerson = function(firstName,lastName)
{
    return{
           firstName:firstName,
           lastName :lastName;
           sayHi :function(){
           return "hiiiiiii";
           }
    };
};


For creating the object only call the factory function and pass the arguments.
Create objects akshay and vicky using factory function:-

Ex.
var akshay = createPerson('akshay' , 'sorate');
var vicky = createPerson('vicky' , 'sorate');
