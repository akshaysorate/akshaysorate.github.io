Difference between constructor and factory function

Constructor

  • constructor is used for creation of object. Constructor main task is intialization
  • constructor automatically returns the values no need to return statement
  • Constructor -which creats the objects then return it.

Using constructor we can create number of same type of objects.Same type of means the objects having same properties.

Syntax:

use capital letter to name the constructor function so that it has easily identify and distinctguish between others

var Constructor_name = function(variables,..,...)
{
.....................;
};


Ex. Create the constructor which has two arguments and it returns the firstname ,lastname

var Person = function(firstName,lastName)
{
  this.firstName : firstName,
  this.lastName : lastName;
};


For creating the object use the keyword 'new' and call the constructor and pass the arguments.
Create objects akshay and vicky using constructor function:-

Ex.
var akshay = new Person('akshay' , 'sorate');
var vicky = new Person('vicky' , 'sorate');
